# Observable

Provides simple tools to implement the "Observer" pattern.

## Installation

The package can be installed by adding `observable` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:observable, "~> 0.3.0"}
  ]
end
```
